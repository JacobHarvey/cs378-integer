vector< vector< bool > > Graph::adjMatrix

Used to store the vertex pairs and the dependencies, size is tasksXtasks large, sets value to 1 at adjMatrix[x][c] if Task x depends on Task c. Can be used to check dependencies or successors by iterating through either rows or columns.
