Has to have the graph initialized, else the function returns false. Evals returns the run order for a list of tasks and their dependencies The Given graph must not have any cycles!!! Must be a DAG graph The graph must follow the guidelines outlined in the sphere page Output: the resulting list of tasks is put into the resultqueue defined within the graph class

vector< vector< bool > > Graph::adjMatrix

Used to store the vertex pairs and the dependencies, size is tasksXtasks large, sets value to 1 at adjMatrix[x][c] if Task x depends on Task c. Can be used to check dependencies or successors by iterating through either rows or columns.
